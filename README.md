# AAS — Associação de Assobiadores de Sumaré

Site estático (HTML/CSS/JS) para a AAS.

## Como abrir

- **Rápido**: dê duplo clique em `index.html`
- **Recomendado (servidor local)**:

```bash
python3 -m http.server 5173
```

Depois abra `http://localhost:5173`.

## O que editar

- `index.html`
  - Links de redes sociais e conteúdo das seções
- `index.html`
  - Textos/agenda/links e seções
- `styles.css`
  - Cores, tipografia e layout

## Estrutura

- `index.html`: página única com seções (Início, Sobre, Eventos, Associe-se, Contato)
- `index.html`: página única com seções (Início, Sobre, Eventos, Contato)
- `styles.css`: estilo e responsividade
- `script.js`: menu mobile e ano automático
- `assets/aas-mark.svg`: marca/ícone
